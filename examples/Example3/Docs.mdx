# `Custom Locale`

This example demonstrates how to use the `@gertvdb/locale` library to create custom locale objects.

<br />

## Demo

<Canvas of={Stories.Standard} />

## Code

```tsx
import { createLocale } from "@gertvdb/locale";

// Create custom locale
const custom = createLocale({ languageOrLocale: 'fr-CA' });
```
