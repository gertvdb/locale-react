# `System Locale`

This example demonstrates how to use the `@tacky-org/locale` library to access the system locale and create custom locale objects.

<br />

## Demo

<Canvas of={Stories.Standard} />

## Code

```tsx
import { SystemLocale } from "@tacky-org/locale";

// Access system locale — always strict, must exist in the dataset
const system = SystemLocale;
```
